# XorShellCode
Xor Encry ShellCode With x86

暂还未开发出命令行传参生成加密文件的方式，生成方式/框架大同小异，只做记录

### 使用:

1. 将CS生成的payload.c放到xor项目中，生成执行，得到加密后代码

2. 将加密代码放到shellcode项目中，生成即可。
